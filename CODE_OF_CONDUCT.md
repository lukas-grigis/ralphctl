# Code of Conduct

> _"I'm a good helper!"_ — Ralph Wiggum

## The short version

Be kind. Be respectful. Help each other out. That's it, really.

## The longer version

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Everyone participating in RalphCTL — issues, pull requests, discussions — is expected to:

- **Be welcoming** — newcomers, experienced developers, and everyone in between
- **Be respectful** — disagreements are fine; personal attacks are not
- **Be constructive** — focus on the problem, not the person
- **Be patient** — this is a side project maintained by humans with lives

Unacceptable behavior includes harassment, trolling, deliberate intimidation, and unwelcome advances of any kind.

## Enforcement

Instances of unacceptable behavior can be reported via [GitHub's private reporting](https://github.com/lukas-grigis/ralphctl/security). Reports will be reviewed and handled appropriately.
